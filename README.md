   Quiz App 
Интерактивное веб-приложение в формате викторины, разработанное на React.
   О проекте
Приложение представляет собой  квиз с системой вопросов и вариантов ответов. Пользователь проходит серию вопросов, после чего получает результат с количеством правильных ответов.
Основной акцент сделан на:
чистую архитектуру компонентов
интерактивность интерфейса
UX (прогресс-бар, мгновенная реакция на действия)
⚙️ Технологии
React (Hooks: useState)
JavaScript (ES6+)
SCSS
Vite
💡 Реализованный функционал
Пошаговое прохождение вопросов
Динамический прогресс-бар
Подсчёт правильных ответов
Условный рендеринг (экран игры / результат)
Обработка пользовательских событий
Перезапуск квиза
